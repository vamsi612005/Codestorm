# Welcome to Neev.AI

Neev.AI is your intelligent personal study companion, designed to help students master subjects through smart quizzes, flashcards, and AI-powered assistance.

## Project info

This project is a modern web application built for students to track progress, take quizzes, and generate study materials.

## Getting Started

To run this project locally, follow these steps:

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)
- MongoDB Database (Cloud or Local)

### Installation

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Step 3: Install the necessary dependencies
npm install

# Step 4: Start the development server (Frontend + Backend)
npm run dev
```

## Technologies Used

This project is built with:

- **Frontend**: Vite, React, TypeScript, Tailwind CSS, shadcn-ui
- **Backend**: Node.js, Express
- **Database**: MongoDB

## Deploying

Building the project for production:

```sh
npm run build
```
